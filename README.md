# Kos-Sign
> 2024 코스콤 신입사원 해커톤 2조 FE

## Branch 전략
### dev branch (default)
개발 브랜치. feature별로 세부 브랜치 나누어 개발 진행 후 develop 브랜치에 merge
### main branch
서버 배포가 진행되는 브랜치. 개발 완료 후 main에 merge 되면 자동 배포(Docker + NCP + Github Actions)
